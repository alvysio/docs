---
title: "Overview of User Permissions"
description: "Quick reference listing every Alvys user permission by category, from rates and dispatch to billing, privacy, and reports, with a short note on each toggle."
---

Please see a list of user permissions below along with a description of what each permission does.

## Rate Permissions

- [ ] View Customer Rate: Ability to view customer rate
- [ ] Edit Customer Rate: Ability to view and edit customer rate
- [ ] View Carrier Rate: Ability to view carrier rate
- [ ] Edit Carrier Rate: Ability to view and edit carrier rate
- [ ] View Trip Value: Ability to view trip value
- [ ] Edit Trip Value: Ability to view and edit trip value
- [ ] Add Accessorial: Ability to add, edit, and remove accessorials

## E-Check Permissions

- [ ] Issue E-Check: Ability to issue e-checks
- [ ] Cancel E-Check: Ability to cancel e-checks
- [ ] Modify E-Check Fee: Ability to make changes to existing e-checks
- [ ] Move Com-check: Ability to move an e-check from one load to another
- [ ] Delete Notes: Ability to delete notes added to a load

## Billing Permissions

- [ ] Billing: Allows the user to view and access the billing loads board
- [ ] Generate Invoice: Ability to generate invoices
- [ ] Pay Driver: Ability to see the pay drivers option in the main menu, and provides access to the pay driver tab and menu option.
- [ ] View Paystubs: Ability to see paystubs within driver profiles
- [ ] Fuel Report: Ability to access the Fuel Report option from the main menu
- [ ] Toll Report: Ability to access the Toll Report option in the main menu

## Dispatch

- [ ] Dispatch: This allows for the user to be considered as a dispatcher within the Dispatchers List in the Assets pages of Alvys
- [ ] Assign Carrier: Ability to assign a carrier to a load
- [ ] Override Stop Status: Ability to override stop statuses
- [ ] Override Load Weight: Ability to override weights set on a load
- [ ] Release Loads: Ability to release the load

## Privacy

- [ ] View Tax ID/SSN: Ability to see the Tax ID/SSN information in the driver profiles, as well as in the carrier profiles
- [ ] Edit Tax ID/SSN: Ability to edit the Tax ID/SSN information in the driver profiles, as well as in the carrier profiles

## General

- [ ] DAT User: Only to be used by companies who utilize a DAT integration. This allows a user to enter the DAT credentials necessary for their company’s DAT integration.
- [ ] TruckStop User: Only to be used by companies who utilize a TruckStop integration. This allows a user to enter the TruckStop credentials necessary for their company’s TruckStop integration.

## Management

- [ ] Add User: Ability to create users
- [ ] Set Permission: Ability to update or change permission settings for other user accounts
- [ ] Activate Carrier: Allows the user to create, activate, or inactivate a carrier.
- [ ] Activate Company: Allows the user to create, activate, or inactivate a company.
- [ ] Edit Carrier: The user may edit an existing carrier, but will not be able to activate or inactivate a carrier without the Activate Carrier permission enabled as well.
- [ ] Edit Company: Ability to edit companies in your actual companies list within tenant (factoring, shipping consignees, etc)
- [ ] Delete Carrier: Ability to delete a carrier company from carriers list. Please use extreme caution when enabling this permission and deleting carriers in Alvys. If the carrier has been assigned to any loads within Alvys, then it will delete them from the load in Alvys as well.
- [ ] Delete Company: Ability to delete a company from companies list. Please use extreme caution when enabling this permission and deleting companies in Alvys. If the company has been assigned to any loads within Alvys, then it will delete them from the load in Alvys as well.
- [ ] Delete User: Ability to delete a user
- [ ] Edit Asset: Ability to edit a trailer, truck, or driver within the assets lists
- [ ] Delete Asset: Ability to delete a trailer, truck, or driver within the assets lists. Please use extreme caution when enabling this permission and deleting assets in Alvys. If the asset has been assigned to any loads within Alvys, then it will delete them from the load in Alvys as well.

## App (Alvys Mobile App Permissions)

- [ ] View Trip Value: Allows driver to view the trip value
- [ ] Issue ECheck: Ability for driver to create and access e-checks
- [ ] Cancel ECheck: Ability for driver to cancel e-checks
- [ ] View Customer Rate Confirmation: Allows driver to view customer rate
- [ ] View Carrier Rate Confirmation: Allows driver to view carrier rate
- [ ] View Payable Amount: Allows driver to see their payable amount

## Load

- [ ] Edit Miles: Ability to edit the loaded and empty miles
