---
title: "Base URL"
description: "Production base URL, path format, and version segment used for all Alvys Public API HTTP requests to integrations.alvys.com endpoints."
---

You can access all our APIs through HTTP requests to URLs:

`https://integrations.alvys.com/api/p/v{version}/<controller>`

For example: `https://integrations.alvys.com/api/p/v1/loads/search`
