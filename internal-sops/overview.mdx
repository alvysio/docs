This is the Implementation team's internal playbook, mirrored from the Notion [Implementation SOP's database](https://app.notion.com/p/8d772d8975f7424683ede0c0b22ffee5). It gives Implementation Managers, Solutions Engineers, and CSMs a single, browsable index of the SOPs, templates, and reference material used to stand up new customers.

Each entry links to its source of truth in Notion or Google Drive — the canonical documents live there, and this index keeps them discoverable from the docs site.

## Segment-specific implementation

The core implementation SOPs are split by customer segment. Start from the SOP that matches the deal:

- **SMB** — [SMB Implementation SOP](https://app.notion.com/2f0a5b0d388a8012b09bd92127a9b105): the streamlined, fastest-safe path to go-live.
- **Mid-Market** — [MM Implementation SOP](https://app.notion.com/2f5a5b0d388a8018b4fbe5655f3bd23a).
- **Enterprise & Strategic** — [ENT/STRAT Implementation SOP](https://app.notion.com/2bda5b0d388a80dba722f1654186789a): the deeper path with multi-location, integration, and phased-rollout considerations.
